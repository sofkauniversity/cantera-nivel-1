Impresión en pantalla

Para poder imprimir en pantalla con PSeInt, basta con usar la palabra Escribir y entre comillas dobles o simples, se escribe el mensaje que se desea mostrar.

Como se observa en la imagen anterior, en la línea 2 y 3, se han escrito dos mensaje, cada uno entre comillas simples y comillas dobles, para demostrar que no hay diferencia alguna.

Después de haber escrito el programa, para poderlo ejecutar, se debe hacer clic en el ícono de Play, como lo indica la flecha roja y el resultado será una consola con el resultado del programa escrito como se muestra en el recuadro rojo.
