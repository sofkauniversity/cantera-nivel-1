Variables

Actualmente para poder almacenar información en programación, se debe crear un espacio en memoria bajo un nombre, dicho espacio en memoria se llama variable.

Las variables en PSeInt, las hay de cuatro tipos:

  • Caracter
  • Real
  • Entero
  • Lógico

Variable tipo Caracter

Las variables de tipo Caracter, son usadas comúnmente para almacenar información como nombres, es decir, cadenas de caracteres, una sola letra e incluso números, con la peculiaridad de que estos números no podrán ser usados en un futuro para temas operacionales como, sumar, restar, multiplicar, dividir, entre otros.

Variables tipo Real

Las variables de tipo Real, son usadas para almacenar números con su parte decimal, tanto negativos como positivos. Por ejemplo la altura de una persona, el cual, podría ser 1.8

Variables tipo Entero

Las variables de tipo Entero, son usadas para guardar número enteros, es decir, números sin su parte decimal. También con la posibilidad de guardar tanto número positivos como número negativos. Por ejemplo, la edad de una persona.

Variables de tipo Logico

Las variables de tipo Logico, tienen la capacidad de guardar solo dos valores, Falso o Verdadero. Por ejemplo, el estado de una persona en un sistema: activo o inactivo.

Como se observa en las imágenes anteriores, cada vez se que quiere definir la creación de una variable, es decir, separar espacio en memoria para almacenar un dato, es importante escribir la palabra Definir, seguido del nombre de la variable, luego se escribe la palabra Como y por último se termina con el nombre del tipo de dato que se desea crear: Caracter, Real, Entero o Logico.
